## 🚀 Quick Start

Each plugin has its own directory with:

- `index.tsx` - The plugin source code
- `manifest.json` - Plugin metadata (name, version, author)
- `README.md` - Detailed documentation and configuration

Navigate to the plugin's folder to see usage instructions.
